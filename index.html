Automated Weather & Logistics ETL Pipeline

A fully automated end-to-end ETL pipeline that pulls real-time weather data from the OpenWeatherMap API every hour for 12 cities across Bangladesh and regional logistics hubs. The pipeline applies business logic transformations — including heat index calculation, composite logistics risk scoring, and Bangladesh season classification — stores results in a SQLite database, and visualises operational risk in a Power BI dashboard. The entire system runs autonomously via Windows Task Scheduler with auto git push after every run.
